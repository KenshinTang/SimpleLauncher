Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => xul\src\main\AndroidManifest.xml
* assets\ => xul\src\main\assets
* libs\gif.jar => xul\libs\gif.jar
* res\ => xul\src\main\res
* src\ => xul\src\main\java\
* src\com\starcor\xul\XulDataNode.aidl => xul\src\main\aidl\com\starcor\xul\XulDataNode.aidl
